# Changelog

## 2.1.4 — 2026-07-17

- установщик больше не загружает jQuery из защищённого каталога модуля и
  работает на нативном JavaScript;
- добавлены кнопка и автоматическая CSPRNG-генерация клиентской ключ-фразы с
  обязательным серверным fallback для пустого поля;
- исправлено включение кнопки установки после принятия лицензии;
- preflight проверяет СУБД, тестовую запись, версии, криптографию, модули
  Bitrix, исходные файлы и все целевые пути до начала установки;
- отсутствие legacy-таблиц допускается, а при наличии полного комплекта
  `sib_kr_*` данные безопасно переносятся в новую схему;
- инструкции, changelog и LICENSE размещаются в корне релизного архива рядом с
  папкой `drdroid.keyrights/`, а не внутри неё.

## 2.1.3 — 2026-07-17

- подтверждены и устранены проблемы аудитов ACL, миграции данных, API,
  установщика, импорта, экспорта и клиентского шифрования;
- дерево и записи фильтруются по правам на сервере, операции перемещения и
  изменения владельца проверяют все затронутые объекты и субъектов;
- добавлена повторяемая миграция `sib_kr_*` → `dr_kr_*` для MySQL/MariaDB и
  PostgreSQL без удаления исходных таблиц;
- параметры API из JSON, GET и POST изолированы от Cookie, некорректный JSON
  отклоняется, ответы получают корректные HTTP-коды и защитные заголовки;
- промежуточное состояние импорта шифруется AES-256-GCM, ограничено по размеру
  и TTL и привязано к администратору, запустившему импорт;
- ключи настроек защищены системной криптографией Bitrix или переменной
  окружения; новые ключи, nonce и пароли создаются CSPRNG;
- удалены внешний jQuery, mcrypt fallback, неиспользуемые ресурсы и отладочные
  остатки; GitHub Actions закреплены по commit SHA;
- добавлены контрактные тесты безопасности, локальная frontend-сборка и
  воспроизводимый релизный архив с SHA-256.

## 2.0.1 — 2026-07-17

Первый публичный установочный релиз модуля `drdroid.keyrights`.

- установочный архив публикуется в GitHub Release;
- архив содержит готовый к установке модуль с корневой папкой
  `drdroid.keyrights/`;
- установщик поддерживает MySQL/MariaDB и PostgreSQL;
- при установке выполняется повторяемая транзакционная миграция ACL из
  `sib_kr_item`/`sib_kr_right` в `dr_kr_item`/`dr_kr_right`, исходные таблицы
  сохраняются;
- закрыты обходы ACL и раскрытие структуры хранилища, усилена валидация API,
  добавлены HTTP-коды ошибок и rate limiting;
- ключи настроек защищаются системным ключом Bitrix или переменной окружения,
  клиентский KDF переведён на PBKDF2-HMAC-SHA256;
- удалены небезопасная десериализация, внешний jQuery, слабые генераторы
  случайных чисел и неиспользуемые ресурсы;
- jQuery 3.7.1 поставляется локально, набор Open Sans сокращён до двух лёгких
  WOFF-начертаний;
- добавлены проверки сборки, контрольная сумма и документация по установке.

## 2.0.0 — 2026-07-17

- первое самостоятельное имя пакета `drdroid.keyrights`;
- обновлены namespace, module ID, component ID и frontend-пути;
- сохранены текущие API, шифрование, миграция legacy-данных и модель прав;
- добавлена MIT-лицензия и репозиторная документация;
- подготовлен установочный релизный архив.
- удалена устаревшая Flash/clipboard-зависимость и заменена на нативное
  копирование в буфер обмена;
- удалён серверный endpoint отправки отчётов, обратная связь перенесена в
  [GitHub Issues](https://github.com/elftorg/keyrights/issues).
